О сайте

Я разработала этот веб-сайт для наработки практических навыков frontend-разработки.

Сайт разработан с помощью HTML, CSS, JavaScript. Форма обрабатывается только на клиентской части с помощью JavaScript(проверка заполненности полей, проверка корректности ввода. На сайте отсутствует серверная часть, соответственно форма нерабочая. Подвал сайта подключен через фрейм. Футер прижат к низу страницы с помощью абсолютного позиционирования для фиксированной высоты футера (решение взято отсюда).

Тематика контента выбрана случайным образом, и не связана с интересами разработчика. Вся информация взята из свободной энциклопедии "Википедия".
